Replace a template element with a new UI:

The container will be replaced by the UI generated.

self.InstanceUI("container_to_replace", "example/ui/example_basic.js");

Template:

<div>
  <div data-id='container_to_replace'>Container content</div>
<div>

To append UIs instead of replace set the Template Mode to self.TEMPLATE_APPEND
