Method Detail

  • $Container()
    Get the component container. If the component is a screen, the container is set using iris-_AbstractScreen#AddScreen function. Otherwise if the component is a UI, the container is set using iris-_AbstractComponent#InstanceUI function. 
    var screenContainer = self.$Container();
  • $Get(p_id)
    Find a JQuery object using its data-id attribute. 
    var $label = self.$Get("span_label");

var rootElement = self.$Get();
    Parameters:
    {String} p_id Optional
    Element data-id value, if no set return root element (optional)
  • Awake()
    Called automatically when a screen is showed. Function to override.
  • CanSleep()
    Called automatically when a screen is going to sleep, before iris-_AbstractComponent#Sleep. If return false, the self.Sleep() function is not called.
  • Create()
    Called automatically only once at the creation phase. Function to override.
  • Destroy()
    Called automatically by iris-_AbstractComponent#DestroyUI. Function to override.
  • DestroyAllUIs(p_idOrJq)
    Remove all UI components from a container. The UIs must be previously created using self.InstanceUI.
    Remove parent references. 
    self.DestroyAllUIs("container");

self.DestroyAllUIs($container);
    Parameters:
    {String|JQuery} p_idOrJq
    UI Container data-id or JQuery object
  • DestroyUI(p_ui)
    Remove a child UI component completely previously created using self.InstanceUI. Remove parent references. 
    var customUI = self.InstanceUI("custom_ui", "custom_ui.js");
self.DestroyUI(customUI);
    Parameters:
    {UI} p_ui
    UI to be removed
  • Hide()
    Hide the template object. If the component is the current screen, this is called automatically after self.Sleep() when iris navigates to other screen. Is called too when a new screen is created. See iris.Goto for more details.
  • InstanceUI(p_id, p_jsUrl, p_uiSettings)
    Create a new UI component instance. The created component is registered into the parent screen or UI, so you must use iris-_AbstractComponent#DestroyUI in order to remove it. The component is added to p_id container or replace it. See iris-_AbstractUI#TemplateMode for more details. 
    self.InstanceUI("custom_ui", "custom_ui.js");

self.InstanceUI("btn_ok", "button.js", {"label":"OK","ico":"accept"});
    Parameters:
    {String} p_id
    Container data-id
    {String} p_jsUrl
    UI file path
    {Object} p_uiSettings
    UI Settings
  • Show()
    Show the template object. If the component is a screen, this is called automatically after self.Awake() when iris navigates. See iris.Goto for more details.
  • Sleep()
    Called automatically when a screen is hidden. Function to override.
