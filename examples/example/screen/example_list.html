Create a group of Instance UIs and add all them to a parent component

The UI generated will be append to the container.

self.InstanceUI("container_to_append", "example/ui/example.js");

Template:

<div>
  <div data-id='container_to_append'>List container content</div>
<div>

Important: The child UI must have its Template Mode set to self.TEMPLATE_APPEND

iris.UI(function (self) {
    self.Create = function() {
        self.TemplateMode(self.TEMPLATE_APPEND);
        self.Template("example/ui/example.html");
    }
}

The default value of Template Mode is self.TEMPLATE_REPLACE
