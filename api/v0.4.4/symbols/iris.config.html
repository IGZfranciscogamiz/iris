Method Detail

  • <static> iris.config.Env(p_env)
    Set or get the current environment. If a configuration file is loaded, iris try set this value using the current URL and the defined environments. If not match anything use "environment-default" The local variables depends of this value. 
    Example: URL=http://localhost:8080/admin
Configuration file:
...
 "environment": {
       "localhost" : "dev"
      ,"www.example.com" : "pro"
 }
...
  
iris.config.Env(); // return "dev"

iris.config.Env("pro"); // set environment to "pro"
    Parameters:
    {String} p_env Optional
    Environment name (optional)
